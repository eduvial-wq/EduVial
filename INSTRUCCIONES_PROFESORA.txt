INSTRUCCIONES PARA ABRIR EL PROYECTO EDU-VIAL
===============================================

Estimada profesora,

Para visualizar correctamente el proyecto de educación vial, siga estos pasos:

1. DESCARGAR EL ARCHIVO:
   - Descargue el archivo "ProyectoEduVial.zip" desde WhatsApp

2. DESCOMPRIMIR EL ARCHIVO:
   - Haga clic derecho sobre el archivo ZIP
   - Seleccione "Extraer todo..." o "Extract all..."
   - Elija una carpeta de destino (por ejemplo: Escritorio)
   - Haga clic en "Extraer"

3. ABRIR EL PROYECTO:
   - Vaya a la carpeta donde extrajo los archivos
   - Busque el archivo "index.html"
   - Haga doble clic sobre "index.html"
   - Se abrirá automáticamente en su navegador web predeterminado

4. NAVEGAR POR EL PROYECTO:
   - En la página principal verá tres secciones:
     * Vial Kids (para 1° ciclo de primaria)
     * Vial Junior (para 2° ciclo de primaria) 
     * Edu-Vial (para secundaria)
   - Haga clic en cualquiera de los botones "Ingresar" para explorar cada sección

IMPORTANTE:
- NO abra el archivo HTML directamente desde Google Drive
- Siempre descargue y extraiga el ZIP primero
- Use un navegador web (Chrome, Firefox, Edge, etc.)

Si tiene algún problema, puede contactar al estudiante para asistencia.
